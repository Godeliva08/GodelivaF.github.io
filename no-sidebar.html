Sales Forecasting by Regression Linear Analysis and A/B Testing of Revamp Landing Page for E-commerce

Role : Data Analyst

Task : Correlation Analysis, Regression Analysis and statistic for hypothesis testing.

Tools : Spreadsheet

Project Description : How to predict future sales by linear regression analysis and defined variables that correlate. Meanwhile, A/B testing was conducted with consideration to z-score and p-value.

Result : The highest correlation was found between freight value and product weight hence I decided to use them to prediction using Regression Linear. The Regression Linear resulted a slope, an intercept and a coefficient of determination to determine the prediction of freight value using product weight.
Meanwhile A/B testing resulted that the null hypothesis could not being rejected due to p-value was higher than the alpha (the significance level) means the revamping page landing could not give an impact significantly.
